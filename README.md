# CoolSpot

A public heat-relief finder for the DC, Maryland, and Virginia region. Locates cooling centers, hospitals, and other air-conditioned facilities alongside Census heat vulnerability and air-conditioning estimates to help residents find cooling during extreme heat events. Built as a static site with a Python data pipeline.

## Data Sources

| Source | Purpose | Coverage |
|--------|---------|----------|
| Census CRE-Heat 2022 | Heat vulnerability tracts | DC, MD, VA |
| Census LACE 2023 | AC availability estimates | Tract-level |
| HealthData.gov anag-cw7u | Hospital locations | US-wide |
| Census ACS 2024 | Demographic context | DC, MD, VA |
| DC HSEMA ArcGIS | Cooling center registry | DC |
| VA VDH WP-Store-Locator | Cooling center registry | VA |
| MD Health Department | Cooling center hub | MD |
| PG County ArcGIS | Cooling centers | MD |

## Disclaimer

This tool is for informational purposes. Always verify facility hours and availability before visiting. In life-threatening heat emergencies, call 911.
